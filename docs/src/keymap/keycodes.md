# Keycodes

Here is the list of the available keycode in RMK.

When used in configuration files, all keycodes are case-insensitive.

## Normal keycodes

| Keycode | Aliases | Usage |
|---------|---------|-------|
| `A`     |         | `A` and `a` |
| `B`     |         | `B` and `b` |
| `C`     |         | `C` and `c` |
| `D`     |         | `D` and `d` |
| `E`     |         | `E` and `e` |
| `F`     |         | `F` and `f` |
| `G`     |         | `G` and `g` |
| `H`     |         | `H` and `h` |
| `I`     |         | `I` and `i` |
| `J`     |         | `J` and `j` |
| `K`     |         | `K` and `k` |
| `L`     |         | `L` and `l` |
| `M`     |         | `M` and `m` |
| `N`     |         | `N` and `n` |
| `O`     |         | `O` and `o` |
| `P`     |         | `P` and `p` |
| `Q`     |         | `Q` and `q` |
| `R`     |         | `R` and `r` |
| `S`     |         | `S` and `s` |
| `T`     |         | `T` and `t` |
| `U`     |         | `U` and `u` |
| `V`     |         | `V` and `v` |
| `W`     |         | `W` and `w` |
| `X`     |         | `X` and `x` |
| `Y`     |         | `Y` and `y` |
| `Z`     |         | `Z` and `z` |

## Special keycodes

| Keycode | Aliases | Usage |
|---------|---------|-------|
| `No`    |         | Ignore this key |
| `Transparent` | `TRNS`, `_` (you can repeat as many times as you like) | Use the next lowest non-transparent key |
